Checklist for Technician:
1.	Plug in the computer so it charges during the match
2.	Plug in the tablet power line to the USB hub
3.	Connect to robot once robot boots up via Ethernet cable (Use Ethernet to USB connector)
4.	Open FRC Driver Station and make sure you have robot code, comms, and joysticks. If not, check Ethernet connection and restart driver station.
5.	Make sure driver station and camera boots up
6.	Check joysticks to make sure the driver joystick is joystick 0
7.	Check tablet and make sure everything works
8.  Check the speaker. Turn the physical knob and make sure the volume is all the way up.
9.	In Setup/Auto tab, set the autonomous play to the correct file depending on the play you would like to run. You can reference the Auto Playbook if you need.
10.	Make sure you are in the correct driver profile
11.	Set the Auto Delay if needed. Talk with your alliance partners to see what their auto does to determine value.
12.	Set Robot Starting Position. Refer to the auto playbook and get measurements from on the field if you need. Once placed, you can click the y switch to use the LL for correct placement if needed.
13.	Go to Tune Tab and make sure everything is green.
14.	Go to TeleOp Tab to get the screen ready for the drivers
15.	Open up the graphs and clear error messages on the Driver Station. Click Prints too so that we record the states in auto/climb modes. Start recording using OBS Studio
